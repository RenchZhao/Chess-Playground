# Chess Master - 国际象棋对弈APP

## 项目简介

这是一个功能完整的国际象棋对弈应用，支持多种对战模式，具有精美的界面设计和强大的AI对手。
开发者借助了kimi K2模型进行了vibe coding实现了这个项目。

## 功能特性

### 对战模式
- **人对弈电脑** - 与智能AI对战，可选择初级、中级、高级难度
- **电脑对弈电脑** - 观看AI自我对局，学习战术策略
- **人对弈人** - 本地双人对战，支持悔棋和和棋

### 核心功能
- 完整的国际象棋规则实现
- 智能AI对手（Minimax算法 + Alpha-Beta剪枝）
- 王车易位、吃过路兵、兵升变等特殊规则
- 将军、将死、逼和自动判定
- 实时走法历史记录
- 游戏统计信息

### 界面特色
- 现代化的响应式设计
- 精美的棋子动画效果
- 直观的拖拽操作
- 优雅的视觉反馈
- 移动端友好

## 使用方法

1. 打开 `index.html` 进入主页面
2. 选择对战模式：
   - 人对弈电脑：选择难度后选择棋子颜色
   - 电脑对弈电脑：选择对局速度
   - 人对弈人：直接进入游戏
3. 在游戏中：
   - 点击棋子选择，再点击目标位置移动
   - 使用控制按钮进行新游戏、悔棋、提和等操作
   - 查看右侧的游戏统计和走法历史

## 游戏规则

### 棋子移动
- **兵**：向前直走一格，斜向吃子，首次可前进两格
- **车**：横向或纵向任意距离
- **象**：斜向任意距离
- **马**：L形移动，可跨越其他棋子
- **后**：兼具车和象的移动方式
- **王**：任意方向一格移动

### 特殊规则
- **王车易位**：王和车同时移动，王移动两格
- **吃过路兵**：兵首次前进两格时，相邻兵可斜向吃掉
- **兵升变**：兵到达对方底线时必须升变为后、车、象或马
- **将军与将死**：王被攻击时必须应将，无法应将即为将死

### 和棋条件
- 逼和：无合法移动且不被将军
- 三次重复局面
- 50步规则
- 子力不足
- 双方同意

## AI算法

### Minimax算法
- 评估函数：棋子价值 + 位置评估 + 王的安全
- 搜索深度：初级2层，中级4层，高级6层
- Alpha-Beta剪枝优化搜索效率

### 位置评估
- 中心控制奖励
- 棋子协调性
- 王的安全评估
- 兵型结构
